# 📊 Adidas US Sales Dashboard (Power BI)

## 🔍 Project Overview

This project presents an interactive sales dashboard built using Power BI to analyze Adidas US sales performance across regions, products, and time.

## 🎯 Objectives

* Analyze sales performance by region, product category, and time
* Identify top-performing products and retailers
* Track monthly trends and profitability

## 📊 Key Features

* KPI Cards: Total Revenue, Total Profit, Total Units, Avg Margin
* Time Analysis: Monthly sales trend
* Regional Analysis: Sales distribution by region
* Product Analysis: Top-performing products
* Retailer Analysis: Revenue contribution by retailers
* Interactive Filters: Year, Product, Sales Method

## 🛠 Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation

## 💡 Key Insights

* West region contributes the highest revenue (~30%)
* Men's Street Footwear is the top-selling product
* Sales peak during mid-year (July–August)
